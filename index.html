Interest

금융은 배웠는데, 내 선택으로 연결되진 않았나요?

지금의 금융 교육은 개념 설명에 머무는 경우가 많습니다. 학생은 중요하다는 말은 듣지만, 왜 중요한지 직접 체감하기 어렵습니다.

학생이 느끼는 문제

  • 금융 개념이 추상적이라 와닿지 않아요.
  • 설명은 들었지만 실제로 어떻게 쓰는지 모르겠어요.
  • 긴 수업보다 짧고 재미있는 방식이 더 집중돼요.

기존 방식의 한계

  • 설명 위주라 금방 지루해집니다.
  • 실제 상황과 연결되지 않아 기억에 남기 어렵습니다.
  • 정답을 맞히는 학습은 있어도 직접 선택하는 경험은 부족합니다.

왜 중요한가요?

금융은 시험을 위한 지식이 아니라, 앞으로의 선택을 위한 기초입니다. 이해만이 아니라 판단 경험까지 있어야 진짜 학습이 됩니다.

Desire

배운 금융을, 내 선택으로 연결하세요

이 서비스는 청소년이 짧은 영상, 퀴즈, 모의투자를 통해 금융 개념을 쉽고 재미있게 배우는 학습 플랫폼입니다.

1

5분 안에 핵심 이해

마이크로 러닝 영상으로 어려운 금융 개념을 짧고 쉽게 배웁니다.

2

바로 확인하는 퀴즈

영상 기반 퀴즈로 배운 내용을 즉시 점검하고 기억에 남깁니다.

3

직접 해보는 모의투자

직접 선택하고 결과를 보는 경험으로 금융 의사결정을 연습합니다.

Core Features

학생은 더 쉽게 배우고, 교사는 더 효과적으로 운영합니다

짧고 쉬운 영상

핵심만 빠르게 전달해 집중력을 지키고, 어려운 개념도 부담 없이 시작할 수 있습니다.

참여형 퀴즈

수동적인 시청에서 끝나지 않고, 바로 참여하면서 이해도를 높입니다.

실전형 시뮬레이션

모의투자로 선택과 결과를 연결해 금융 감각을 기를 수 있습니다.

학교 적용 가능성

학생 참여와 학습 효과를 동시에 높일 수 있어 수업 도구로도 활용하기 좋습니다.

Expected Change

이 서비스를 사용하면 이렇게 달라집니다

이해

추상적인 금융 개념이 실제 사례와 연결되어 더 쉽게 이해됩니다.

참여

짧은 영상과 퀴즈로 끝까지 참여하는 비율이 높아집니다.

경험

모의투자를 통해 금융 의사결정을 직접 경험할 수 있습니다.

Action
